# DataSurgeon (WIP)
DataSurgeon (ds) is a tool designed for Incident repsonses, Pentests and CTF challenges

# Project Goals
* Able to recieve input from standard output
* Argparse support
* CSV output

# Current Extraction Features
* Emails
* Credit Cards
* URL's
* IP Addresses
* MAC Addresses
* SRV DNS Records
